What gets printed as a result of running the program in the previous step? 

public class VoidMethodsNoParams {

public static void m1 ( ) {
	System.out.println (“A”);
}

public static void m2 ( ) {
	System.out.println (“B”);
}

public static void m3 ( ) {
	System.out.println (“C”);
	m2 ( );
}

public static void main (String [ ] args) {
	m1 ( );
	m3 ( );
}
}
AC
Nope. Notice that the method m3 is calling m2 inside it.
ABC
No. Look closely at the method m3: it first prints out "C" and afterwards calls m2.
ACB
Correct!
