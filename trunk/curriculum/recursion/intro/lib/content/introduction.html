What is recursion?

What is recursion? It's a programming technique in which a method calls itself. For example:

public static void recur ( ) {
	System.out.println ("This is recursion!");
	recur ( );
}

In the method recur, notice the second line: there is a call to the recur method from within the recur method itself. We refer to the call as a recursive call.

In this case, a call to recur produces fairly useless results. It will produce the output: 

This is recursion!
This is recursion!
This is recursion!
This is recursion!
This is recursion!
This is recursion!

on and on and on and on...
