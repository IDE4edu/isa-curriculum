A base case is used to stop a recursion

As we noted in the previous activities, we need a way to get the method to stop calling itself. This involves some kind of conditional, where the method will choose one of two (or more) different cases each time it is called:

		if (we shouldn’t make a recursive call) {
			... do some things ...  
		} else { 
			... 
			make a recursive call; 
			...
		}

The “do some things” code is referred to as a base case. The other code run in the else condition is a recursive case.

Depending on what your method needs to do, there may be more than one base case, and more than recursive case. Remember, if there isn't at least one base case, the method will never stop!
