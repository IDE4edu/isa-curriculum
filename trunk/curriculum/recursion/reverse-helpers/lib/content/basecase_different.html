The base case is different

The error is that reverse("?") should be handled differently from reverse("doesthiswork?"). A one-character string in this situation is referred to as a base case. The reverse method should not make a recursive call when given a one-character string; instead, it should do some computation, then return to its caller.

Every correct recursive method will include one or more base cases. Every recursive call should provide arguments that get closer to the base cases. Omitted or faulty base case handling most likely results in an infinite recursion.
