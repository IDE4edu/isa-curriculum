We can take advantage of similarities in consecutive definitions, as follows: 

public String reverse932 (String s) {
   return reverse931 (allButFirst(s)) + s.substring (0, 1); 
}

public String reverse931 (String s) {
   return reverse930 (allButFirst(s)) + s.substring (0, 1); 
}

Those two methods are specialized—each only works on a string of a given length. But by removing the underlined digits, we can produce a general reversal method that almost works on all strings. Here's the code. 

public String reverse (String s) {
   return ( reverse(allButFirst(s)) + s.substring(0,1) );
}
